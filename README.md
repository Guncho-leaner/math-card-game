# 🎮 どうぶつさんすうカード (Animal Math Card Game)

小学生（主に1年生）向けの一桁の足し算・引き算を楽しく学べる、**100%完全無料・完全オフライン対応・音声入力対応**のスマートフォン向け算数カードゲームです。

かわいい動物たち（うさぎ、パンダ）と一緒に、声に出して楽しく10問全問正解を目指しましょう！

---

## 🌟 主な特徴

1. **🎤 直感的な音声入力回答**:
   - Web Speech API の SpeechRecognition（音声認識）を採用。キーボード入力や画面タップが難しい年齢のお子様でも、声に出して直感的に回答できます。
2. **🔊 完全ローカル高品質オーディオ**:
   - 有料の外部クラウドAPI（Google CloudやOpenAIなど）は一切使用せず、Macのクリアなシステム音声（Kyoko）で事前にレンダリングした**計130個以上の高音質WAV音声ファイルを内蔵**しています。
   - ネット接続なし（オフライン）や電波の悪いお出かけ先、車の移動中でも、美しく滑らかなイントネーションで問題や褒め言葉・応援の言葉を喋りかけてくれます。
3. **🎨 極上のサンリオ風ちびキャラSVG**:
   - ぷっくりとした3D調グラデーション、ふわっとしたエアブラシ風のRosyチーク、キラキラしたつやのある瞳など、洗練された愛らしいキャラクターイラスト（うさぎ・パンダ）を完全独自ベクター実装。
   - 正解すると腕をあげて大喜び（星やハートのキラキラが飛び交い、うさぎちゃんは耳が折れ曲がるギミック付き）、間違えると汗をかいて驚くなど、表情が豊かに動きます。
4. **📱 スマートフォン・タブレット最適化**:
   - HSLカラーパレットを使用した、明るく目に優しいガラスモフィズム（Glassmorphic）デザイン。
   - ぷっくりとした押しやすい3D風ボタンや、スムーズなモーダルアニメーション（最新のCSS `@starting-style` 使用）。
   - 静かな場所で遊ぶためのテンキー（数字パッド）のフォールバック機能も搭載。

---

## 🚀 遊び方とレベル設定

1. **スタート！** を押す。
2. **たしざん** または **ひきざん** を選ぶ。
3. 学期（レベル）を選ぶ：
   - **1がっき**: こたえが 10まで（引き算は 10以内）
   - **2がっき**: こたえが 20まで（引き算は 20以内）
   - **3がっき**: こたえが 20いじょう（引き算は 答えが20以上）
4. 問題カードが出たら、自動でマイクが起動します。大きな声で答えを言ってみましょう！（例：「ご！」「じゅうに！」）
5. 10問終了後にスターの獲得数と最終結果が表示されます。

## 🌐 GitHub Pages へのデプロイ方法（スマートフォンで遊ぶために）

スマホのブラウザでマイク（音声認識機能）を安全に動作させるには、**`https://`（セキュアな通信）**のURLが必要です。GitHub Pagesを利用することで、誰でも一瞬でHTTPSのゲーム専用URLを無料公開できます。

### 1. ローカルでの Git 初期化とコミット（当フォルダで実施）
```bash
git init
git add .
git commit -m "First commit: どうぶつさんすうカード (プレミアムイラスト&高品質ローカル音声版)"
```

### 2. GitHubに新しいリポジトリを作成
1. [GitHub](https://github.com/) にログインし、右上「＋」ボタン ➔ **New repository** を選択。
2. リポジトリ名（例: `math-card-game`）を入力し、**Create repository** をクリック。
3. 表示されたコマンドのうち、以下を実行して push します：
   ```bash
   git branch -M main
   git remote add origin https://github.com/あなたのユーザー名/math-card-game.git
   git push -u origin main
   ```

### 3. GitHub Pages の有効化
1. 作成したリポジトリのページで **Settings**（設定）タブを開きます。
2. 左メニューから **Pages** を選択します。
3. **Build and deployment** セクションの **Branch** で、`None` から `main` に切り替え、`/ (root)` のまま **Save**（保存）を押します。
4. 数分待つと、ページ上部に専用のHTTPSリンクが表示されます！
   （例: `https://あなたのユーザー名.github.io/math-card-game/`）

このURLをスマホで開くだけで、マイク音声回答を含めたすべての機能が動作するようになります！📱🎉
